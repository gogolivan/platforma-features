
# Firewall

A feature to enable firewall 

## Example Usage

```json
"features": {
    "ghcr.io/gogolivan/platforma/platforma-features/firewall:1": {
        "github": true
    }
}
```

## Options

| Options Id | Description | Type | Default Value |
|-----|-----|-----|-----|
| github | Enable GitHub | boolean | true |
